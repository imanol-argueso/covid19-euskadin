Sobre esta web

Whatsapp Facebook Twitter

Este sitio web se actualiza de forma automática con los datos sobre la evolución del covid-19 de Open Data Euskadi, el portal de datos abiertos del Gobierno Vasco.

En Open Data Euskadi se publica un dataset que se nutre del boletín diario del departamento de Salud del Gobierno Vasco. Este dataset contiene datos con diferente frecuencia de actualización (diaria, semanal, etc.). Es por ello que algunos datos de este sitio web se actualizan diariamente y otros con una frecuencia mayor.
